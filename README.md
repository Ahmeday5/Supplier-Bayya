# Bayya Supplier Dashboard

The **Supplier Dashboard** for the **Bayya platform**, built to help suppliers manage their daily operations through a clean, scalable, and user-friendly interface.

It is part of a complete ecosystem that includes a customer app and an admin dashboard.

---

## 🧩 Overview

Suppliers can:
- Log in using their phone number
- Manage orders across all statuses
- Control products, pricing, and availability
- Define delivery areas and stations
- Track financial balances and reports
- View customer ratings and feedback

Built with **Angular 18 Standalone Components**, focusing on performance and responsive design.

---

## 🚀 Key Features

### 📦 Orders
- Full order lifecycle management  
- Status flow: Pending → Confirmed → Shipped → Delivered  
- Status-based filtering  
- Returns and refunds tracking

### 🛒 Products
- Manage supplier products (edit, delete, publish/unpublish)
- Search and pagination
- Add products from the global catalog with custom pricing and limits
- Multiple suppliers can list the same product independently

### 🚚 Delivery
- Manage delivery areas
- Add delivery stations from a central database

### 💰 Finance & Reports
- Wallet balance and dues tracking
- Account statements, penalties, and returns reports

### ⭐ Ratings & Account
- View customer ratings
- Profile management, logout, and account deletion

---

## 🎨 UI & UX

- Interactive and collapsible sidebar
